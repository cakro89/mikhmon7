### 🛠️ untuk install di docker bisa compose fle berikut

For basic compose :


1. docker run:
   ```bash
   docker pull cakro8/mikhmon7
   ```
 

2. docker run:
   ```bash
   docker run --name mikhmon7 --restart always -p 8001:80 cakro89/mikhmon7
   ```
untuk port 8001 bisa kalian rubah sendiri
